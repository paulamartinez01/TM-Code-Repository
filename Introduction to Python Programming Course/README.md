# PennyWise, Your Personal Budget and Expenses Tracker

## General Information
Pandemic-era or not, personal budgeting has never been an easy. PennyWise is a simple automated budget tracker that aims to instill discipline to any user in the
context of budgeting and allocation of money. While PennyWise may be considered a prototype relative to the more advanced budget trackers, it is a first step 
towards financial prudence and independence.

## Room for Improvement
